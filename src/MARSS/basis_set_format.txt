Basis sets should be saved as .mat files and contain the three structures below:
metabolites, header, and artifacts. Metabolite names should follow standard 
conventions with the exception of 2HG, twoHG. Names provided to the simulator 
must match (case insensitive) the names in the basis set file. If the fshift is 
omitted from a metabolite, a default value of 5Hz is assigned in the model. When 
fshift is zero, it is redefined to +/-1e-10.

Optional fields are indicated by *.

Written by: Ing. John T LaMaster 13.10.2022

>>> Metabolites
                 ace: [1x1 struct]
                 ala: [1x1 struct]
                  ⋮        ⋮

>>> Metabolites.metab
                  fid: [2×8192 double]  # normal FID
                  min: 0                # wrt Cr
                  max: 3.5              # wrt Cr
              *fshift: 10               # Hz
             *fid_OFF: [2×8192 double]  # J-difference edit OFF

>>> Header
                   B0: 3                # T
    carrier_frequency: 127.7324         # MHz
            dwelltime: 2.5000e-04       # s
            linewidth: 1                # Hz
                notes: ''               # Brief description of the basis set
                   Ns: 8192             # points
                  ppm: [1×8192 double]  # ppm
                   TE: 144              # ms
               vendor: 'GE'
                    t: [1×8192 double]  # s
        spectralwidth: 2000             # Hz
           centerFreq: 4.6500           # ppm
           
>>> Artifacts
                 phi0: [1×1 struct]     # Zero-order phase
                 phi1: [1×1 struct]     # First-order phase
               fshift: [1×1 struct]     # Global frequency shift
                    d: [1×1 struct]     # Metabolite Lorentzians
                  dmm: [1×1 struct]     # MM/Lip Lorentzians
                    g: [1×1 struct]     # Metabolite Gaussian
                  gmm: [1×1 struct]     # MM/Lip Gaussian
                  snr: [1×1 struct]     # SNR (dB)
                   b0: [1×1 struct]     # Mean B0 inhomogeneity offset
               b0_dir: [1×1 struct]     # B0 spatial difference range
          *multi_coil: [1×1 struct]     # Used to scale transient SNRs
           *coil_sens: [1×1 struct]     # Coil sensitivity scaling factors
         *coil_fshift: [1×1 struct]     # Coil frequency drift
           *coil_phi0: [1×1 struct]     # Coil phase drift
               
>>> Artifacts.artifact
                  min: -45.0            # deg
                  max:  45.0            # deg
